## Crash report

### perfmon

Run cmd as admin:

```bash
perfmon/report
```

### winstat

Run cmd as admin:

```bash
winstat prepop
```

Open up the Action Center, you can use the “View reliability history” link to get there, but we’d recommend opening the start menu or start screen and searching for “reliability”, which is probably faster.
